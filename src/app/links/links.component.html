There are several kinds of links:

  • Internal links within the PDF files.
  • External links on web server (usually with an https URL).
  • Links to file on the local PC or the local network. The PDF viewer doesn't support such links.

By default, external links are opened in the same window. In other words, clicking the link amounts to leaving the Angular application. To prevent this, you have to options:

  • Add the target when you're creating the PDF file. That's generally the best solution.
  • Use the defaultOptions to override the default for externalLinkTarget. The source code on the right-hand side shows you how to do it.
